# Customer Segmentation Using RFM & K-Means Clustering

## Deskripsi Proyek
Proyek ini melakukan segmentasi pelanggan menggunakan **analisis RFM (Recency, Frequency, Monetary)** dan model **K-Means clustering**. Segmentasi ini membantu perusahaan ritel e-commerce internasional untuk:

- Memahami perilaku pelanggan  
- Mengidentifikasi pelanggan paling berharga  
- Merancang strategi pemasaran yang lebih tepat sasaran  

Dataset yang digunakan berasal dari [Online Retail II dataset (Kaggle)](https://www.kaggle.com/datasets/mashlyn/online-retail-ii-uci), yang berisi data transaksi penjualan dari perusahaan ritel e-commerce internasional.

## Persiapan Dataset
Dataset memiliki 1.067.371 baris dan 8 kolom, antara lain:
- `Invoice` : kode faktur transaksi
- `StockCode` : kode produk
- `Description` : deskripsi produk
- `Quantity` : jumlah produk yang dibeli
- `InvoiceDate` : tanggal transaksi
- `Price` : harga per unit
- `Customer ID` : ID pelanggan
- `Country` : negara pelanggan
